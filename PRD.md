# PRD: 럭셔리 호텔 웹사이트

## 1. 프로젝트 개요

### 1.1 프로젝트명
**Luxe Haven Hotel** - 프리미엄 럭셔리 호텔 웹사이트

### 1.2 프로젝트 목표
파라다이스시티(p-city.com)를 레퍼런스로 한 고급 호텔 예약 웹사이트 구축. 세련된 UI/UX와 직관적인 예약 시스템을 통해 사용자에게 프리미엄 경험 제공.

### 1.3 기술 스택
- **Frontend Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **State Management**: Zustand
- **Animation**: Framer Motion
- **Date Picker**: react-day-picker
- **Image Optimization**: Next.js Image
- **Icons**: Lucide React

## 6. 인터랙션 및 애니메이션

### 6.1 페이지 전환
- 부드러운 페이드 인/아웃
- 스크롤 기반 애니메이션 (Intersection Observer)

### 6.2 호버 효과
- 카드: scale(1.02) + shadow 증가
- 버튼: 배경색 변화 + 미세한 lift
- 이미지: 줌 인 효과
